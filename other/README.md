# Onboarding and settings

Recommend default settings for Visual Studio Code [(settings.json)](./etc/skel/.local/share/code-server/User/settings.json) and a other short onboarding readme [(README.md)](./etc/skel/projects/README.md) in the projects folder.

> Add the folder `/etc/skel` to your Linux server, to automatic create the projects folder with the readme for new users.
